# CLI-Finance
<img width="1512" height="390" alt="Image" src="https://github.com/user-attachments/assets/d08b67ab-c856-479e-bd2a-dd068ed960ae" />

A CLI-based personal finance tracker for logging expenses and visualizing monthly spending.

## Installation

```bash
git clone https://github.com/IoannisDev/Cli-Finance.git
cd Cli-Finance
pip install -r requirements.txt
python cli.py
```
