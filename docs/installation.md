# Installation

## Stable release

To install CLI Finance, run this command in your terminal:

```sh
uv add CLI-Finance
```

Or if you prefer to use `pip`:

```sh
pip install CLI-Finance
```

## From source

The source files for CLI Finance can be downloaded from the [Github repo](https://github.com/IoannisDev/CLI-Finance).

You can either clone the public repository:

```sh
git clone https://github.com/IoannisDev/CLI-Finance
```

Or download the [tarball](https://github.com/IoannisDev/CLI-Finance/tarball/main):

```sh
curl -OJL https://github.com/IoannisDev/CLI-Finance/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd CLI-Finance
uv sync
```
